# AI Recommendation System

## Project Overview

This project is a simple AI Recommendation System developed using Python. The system recommends courses to users based on their interests by matching user preferences with predefined course categories. It demonstrates the basic concepts of recommendation systems and decision-making logic in Artificial Intelligence.

## Features

* Recommends courses based on user interests
* Simple and interactive command-line interface
* Uses Python data structures and conditional logic
* Easy to customize with new courses and categories

## Technologies Used

* Python
* Google Colab / Jupyter Notebook

## Working

1. The user enters their area of interest.
2. The system compares the input with available course categories.
3. Matching courses are identified.
4. Personalized course recommendations are displayed to the user.

* Decision-Making Logic
